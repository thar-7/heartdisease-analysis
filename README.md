# ❤️ Heart Disease Prediction Using Machine Learning

## Overview

Heart Disease Prediction is a machine learning-based web application that predicts the likelihood of heart disease using patient health data. The system analyzes medical parameters such as age, gender, blood pressure, cholesterol, blood sugar, heart rate, and ECG results to provide an early risk assessment. This project aims to support healthcare professionals and individuals by enabling faster and more informed decision-making.

## Features

* Predicts the risk of heart disease using machine learning models.
* Simple and user-friendly web interface.
* Real-time prediction based on patient input.
* Data visualization for better analysis.
* Optional patient record storage using MongoDB.
* Easy to deploy and extend for future enhancements.

## Tech Stack

* **Language:** Python
* **Machine Learning:** Scikit-learn, Pandas, NumPy
* **Visualization:** Matplotlib, Plotly
* **Backend:** Flask
* **Frontend:** HTML, CSS, JavaScript
* **Database:** MongoDB
* **Development Tools:** Jupyter Notebook, Visual Studio Code
* **Version Control:** Git & GitHub

## Machine Learning Models

* Logistic Regression
* Decision Tree
* Random Forest

## Applications

* Early heart disease risk prediction.
* Clinical decision support for healthcare professionals.
* Preventive health monitoring.
* Educational project for machine learning in healthcare.

## Future Enhancements

* Integration with IoT health monitoring devices.
* Deep learning models for improved prediction accuracy.

## Conclusion

This project demonstrates the practical application of machine learning in healthcare by providing an efficient and reliable system for heart disease risk prediction. It highlights how AI can assist in early diagnosis, improve healthcare outcomes, and support informed medical decision-making.
